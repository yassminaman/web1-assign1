HTML Tags Reference

em
This tag stands for emphasis. It is used to give special importance or stress to a word or phrase in a sentence.
img
This tag is used to display an image on a web page.
ul
This tag stands for Unordered List. It is used to create a list of items with bullet points.
select
This tag is used to create a drop-down list that allows users to choose one option from multiple choices.
section
This tag is used to group related content together into a separate section of a webpage.
